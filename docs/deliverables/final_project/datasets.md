# Example Datasets for the Final Project

You can use the curated sources below to identify data for your final project. The lists emphasize open licensing, clear documentation, and spatial usability. All items are intended to be directly usable in Python/GIS workflows. You can use any other publicly available datasets.

---

## Quick Tips

- Check licensing/terms; cite source and access date.
- Keep metadata/data dictionaries with your downloads.
- Save raw and processed files separately; script transformations.
- Record CRS for spatial layers; align before joins.
